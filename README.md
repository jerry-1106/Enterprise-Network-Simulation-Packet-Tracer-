# Enterprise Network Simulation (Packet Tracer)

## Overview
Enterprise-style network simulation built using Cisco Packet Tracer to demonstrate core networking concepts and configurations.

## Project Scope
- Designed a multi-segment enterprise network
- Implemented VLAN-based network segmentation
- Configured routing between subnets
- Validated connectivity with end-to-end testing

## Key Configurations
- VLAN creation and assignment
- Inter-VLAN routing
- IP addressing and subnetting
- Connectivity verification using ICMP

## Tools & Technologies
- Cisco Packet Tracer
- TCP/IP
- VLANs
- Routing

## Files
- Packet Tracer simulation file (.pkt)
